================================================================================
PLATFORM-SPECIFIC ARCHITECTURE
================================================================================

1. DATABASE LAYER
   ────────────────────────────────────────────────────────────
   
   DESKTOP (Tauri):
   • SQLite stored at ~/.sage/users/{uid}/sage.db
   • Per-user database via bindUserId(uid)
   • Schema: tasks, messages, files, sessions, settings, sync_queue
   • Async schema initialization on each user bind
   • Direct file access for attachments
   
   iOS (Capacitor):
   • IndexedDB in WebView (database name: "sage")
   • Same schema as SQLite, adapted to IndexedDB transactions
   • ~50MB storage limit (varies by iOS version)
   • No persistent data on app uninstall/cache clear
   
   WEB:
   • Same IndexedDB as iOS
   • Similar storage quotas
   • Browser-dependent persistence

2. SETTINGS STORAGE
   ────────────────────────────────────────────────────────────
   
   DESKTOP:
   • Primary: SQLite settings table
   • Fallback: localStorage
   
   iOS/Web:
   • Primary: localStorage only
   • Note: MCP/skills paths saved but NOT used
   
   Critical: User switching clears in-memory cache, forces reload

3. CLOUD SYNC ARCHITECTURE
   ────────────────────────────────────────────────────────────
   
   All platforms:
   • Fire-and-forget: local write → enqueue → worker loop
   • Retry queue (SQLite on desktop, IndexedDB on iOS)
   • Exponential backoff: 5s → 15s → 45s → 2m → 5m → 30m
   • 5-second tick interval, batches of 10 items
   • Operations: messages.insert, tasks.upsert, files.upsert
   
   iOS specific:
   • Same retry logic works with IndexedDB store
   • Survives app pause/resume
   • Max queue size limited by IndexedDB quota

4. AUTHENTICATION
   ────────────────────────────────────────────────────────────
   
   DESKTOP:
   • OAuth → system browser → deep link sage:// → callback
   
   iOS:
   • OAuth → ASWebAuthenticationSession → ai.sage.app:// → callback
   • Deep link listener: App.addListener('appUrlOpen', ...)
   • Fallback polling if deep link not fired
   
   WEB:
   • OAuth → in-page redirect → auto-detect via URL hash

5. FILE HANDLING
   ────────────────────────────────────────────────────────────
   
   DESKTOP:
   • Filesystem paths: ~/.sage/users/{uid}/sessions/{sessionId}/attachments/
   • Direct file I/O via Tauri plugins
   
   iOS/Web:
   • No filesystem access
   • Attachments: Base64 strings in IndexedDB
   • Size inflation: +33% due to base64 encoding
   • Max file size: Limited by IndexedDB quota

KNOWN ISSUES
  🚨 IndexedDB quota limit (~50MB)
     Impact: Large sessions with many attachments may fail
     Severity: MEDIUM
     Fix: Monitor quota, implement cleanup strategy
  
  🚨 Data loss on app cache clear
     Impact: User loses all local data if cache is cleared
     Severity: MEDIUM
     Fix: Force cloud sync on app resume
  
  🚨 Session aggregation performance
     Impact: Large sessions (100+ messages) may cause jank
     Severity: LOW
     Fix: Paginate/lazy-load messages
  
  🚨 OAuth deep link timeout
     Impact: If user closes browser, auth hangs
     Severity: LOW
     Fix: Polling fallback (already implemented)
  
  🚨 MCP/Sandbox settings saved but ignored
     Impact: User may configure invalid settings
     Severity: LOW
     Fix: Hide these options on iOS UI

================================================================================
PLATFORM DETECTION MECHANISM
================================================================================

Priority Order:
  1. Check isTauri → '__TAURI_INTERNALS__' in window
  2. Check isCapacitor → window.Capacitor?.isNativePlatform?.()
  3. Fallback isWeb → true if neither above

Mobile Detection:
  • isCapacitor always = mobile
  • OR window.innerWidth < 768 (web breakpoint)

Code Pattern:
  import { isTauri, isCapacitor, isWeb, isMobile } from '@/shared/lib/platform';
  
  // Desktop-only operations
  if (isTauri) { /* filesystem, native APIs */ }
  
  // Mobile-only operations
  if (isCapacitor) { /* Capacitor plugins */ }
  
  // Browser-specific
  if (isWeb) { /* web APIs */ }
