# 华泰研究所周度行业观点查询

查询华泰研究所最新周度行业/赛道观点，输出基本面、短期交易、参考思路或赛道意见。

## 核心原则

1. **只返回最新一条** - 有日期则返回该日期及之前的最新一条，无日期则返回最新一条
2. **行业观点仅输出 3 个字段** - 基本面 + 短期交易 + 参考思路（相对沪深300）
3. **赛道观点仅输出赛道意见**
4. **完全保持数据原文** - 不做任何修改、精简、总结或重新表述
5. **实时调用** - 每次查询都从接口实时获取数据

## 适用场景

- **查询行业观点**：针对具体一级行业（如"煤炭"、"食品饮料"、"银行"等）查询其周度观点，输出基本面、短期交易、参考思路 3 个字段
- **查询赛道观点**：针对大类赛道（如"消费"、"金融地产"、"TMT"等）查询其周度观点，输出赛道意见字段

## 数据源说明

数据源为华泰研究所行业周度观点数据。

### API Key 配置

API Key 通过环境变量 `HTSC_APP_KEY` 配置。

### 日期参数规则

- 有日期则返回**小于等于该日期的最新一条**，无日期则返回**最新一条**
- 支持格式：`YYYY-MM-DD`、`YYYY/MM/DD`、`YYYY年MM月DD日`、`YYYYMMDD`

## 触发条件

- 行业相关："食品饮料行业怎么看"、"银行板块观点"
- 赛道相关："消费赛道观点"、"TMT赛道分析"
- 查询对象明确："XX行业"、"XX赛道"

## 输出格式

### 行业观点

```

### 基本面
（原文）

（原文）

### 参考思路（相对沪深300）

数据来源：华泰证券研究所行业周度观点-{日期}
```

### 赛道观点

```
数据来源：华泰证券研究所行业周度观点-{日期}
```

## 风险揭示

每次输出内容末尾必须附加以下风险揭示：

> 以上内容由 AI 基于华泰证券研究报告及其附属成果总结生成，可能省略完整报告中的重要信息、市场风险、业绩不及预期等风险，请务必阅读完整原始报告。有关数据、观点、评级等具有时效性，请自行核实最新信息。以上内容仅供研究学习使用，不构成任何投资建议。请勿二次传播并自行承担投资风险。

## 查询处理逻辑

1. **识别意图**：区分行业观点与赛道观点
   - 行业观点：用户提到具体行业名称（如"煤炭"、"食品饮料"）或含"行业"、"板块"等词
   - 赛道观点：用户提到赛道名称（如"消费"、"TMT"）或含"赛道"、"大类"等词
2. **提取参数**：提取查询对象和日期（可选）
3. **验证范围**：检查行业/赛道是否在支持列表内，详见 `references/available-sectors.md`
4. **调用 API**：使用 `HTSC_APP_KEY` 构建请求并获取数据
5. **输出结果**：按格式输出，仅保留最新数据，保持原文不变

## 错误处理

- **API Key 未配置**：未识别到您的API KEY，请前往行知 App 或行知网页版，进入"华泰智研 Skill 及 MCP 专区"，点击"API KEY"查看并复制您的专属认证码
- **查询对象不在数据源中**：提示不在覆盖范围内，建议查看 `references/available-sectors.md`
- **多个匹配结果**：提供所有匹配项
- **API 请求失败**：提示具体错误信息（超时、连接失败、HTTP 错误等）
- **时间日期格式错误**：提示用户确认查询日期格式
