# HTSC Valuation Skill

## 概述

本 skill 对接华泰研究所估值模型接口，提供以下能力：
- 根据公司名称查询并下载估值模型 Excel 文件
- 查询利润表、资产负债表、现金流量表完整数据
- 查询指定财务指标（如营业收入、归母净利润、净利率等）

## 环境要求

- Python 3.8+
- 依赖：`requests`
- 环境变量：`HTSC_APP_KEY`（必须设置）

## 处理流程

### 1. 检查环境变量

调用脚本前，确认环境变量 `HTSC_APP_KEY` 已设置。该值将作为请求头 `Authorization` 的值发送。如未设置，提醒用户：
> 未识别到您的API KEY，请前往行知 App 或行知网页版，进入“华泰智研 Skill 及 MCP 专区”，点击“API KEY”查看并复制您的专属认证码

### 2. 识别用户意图

根据用户请求判断调用哪个功能：

| 用户意图 | 调用方法 | 参数 |
|---------|---------|------|
| 下载估值模型文件 | `queryValuationFile` + `downloadFile` | `secuAbbr`（公司简称） |
| 查询三表数据（利润表/资产负债表/现金流量表） | `queryValuationData` | `secuAbbr`, `indexAbbr`（表全名） |
| 查询指定指标 | `queryValuationData` | `secuAbbr`, `indexAbbr`（指标名，支持模糊匹配） |

### 3. 调用脚本

使用 `scripts/htsc_valuation.py` 执行具体请求：


命令说明：
- `file <公司简称>` — 查询估值模型文件信息（返回 gatewayFileId、文件名、研报标题、外发时间）
- `data <公司简称> <表名或指标名>` — 查询估值数据
- `download <fileId> <保存路径>` — 下载估值模型文件

### 4. 结果格式化

脚本返回的 JSON 结果需按以下规则展示给用户：

1. **数据来源标注**：所有结果顶部必须标注：`数据来源：华泰证券研究所`
2. **研报关联信息**：如有 `title` 和 `pubDate`，必须展示：
   - 模型关联研报标题：{title}
   - 研报外发时间：{pubDate}
3. **数据完整性**：指标数据必须完整展示，不得截断
4. **编码**：确保 UTF-8 编码，中文不乱码
5. **风险揭示**：脚本输出末尾固定包含风险揭示，必须完整原样返回，不允许总结或篡改。内容如下：
   > 风险揭示：以上内容由 AI 基于华泰证券研究报告及其附属成果总结生成，可能省略完整报告中的重要信息、市场风险、业绩不及预期等风险，请务必阅读完整原始报告。有关数据、观点、评级等具有时效性，请自行核实最新信息。以上内容仅供研究学习使用，不构成任何投资建议。请勿二次传播并自行承担投资风险。

## 注意事项

- `indexAbbr` 传入表名时必须使用全名：`利润表`、`资产负债表`、`现金流量表`
- `indexAbbr` 传入指标名时支持模糊匹配，如：`净利润`、`营业收入`
- 接口返回的 `valueType`：`1` 表示实际值，`2` 表示预测值
- 所有请求和响应均使用 UTF-8 编码
- 请求头 `Authorization` 的值从环境变量 `HTSC_APP_KEY` 获取
