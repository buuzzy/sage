# HTSC Daily Review Skill

## 概述

本 skill 对接华泰研究所每日复盘接口，提供以下能力：
- 查询最近一个交易日的 A 股复盘数据
- 查询最近 5 个交易日的 A 股复盘数据

## 环境要求

- Python 3.8+
- 依赖：见 [requirements.txt](requirements.txt)，安装命令 `pip install -r requirements.txt`
- 环境变量：`HTSC_APP_KEY`（必须设置）

## 处理流程

### 1. 检查环境变量

调用脚本前，确认环境变量 `HTSC_APP_KEY` 已设置。该值将作为请求头 `Authorization` 的值发送。如未设置，提醒用户：
> 未识别到您的API KEY，请前往行知 App 或行知网页版，进入“华泰智研 Skill 及 MCP 专区”，点击“API KEY”查看并复制您的专属认证码


### 2. 识别用户意图

根据用户请求判断查询范围：

| 用户意图 | queryType | 说明 |
|---------|-----------|------|
| 最近一个交易日 / 今日 / 昨天 | `1` | 获取最近一个交易日的复盘 |
| 最近一周 / 最近5个交易日 / 近期 | `2` | 获取最近5个交易日的复盘 |

### 3. 调用脚本

使用 `scripts/htsc_daily_review.py` 执行具体请求：


命令说明：
- `recent` — 查询最近一个交易日的复盘数据
- `week` — 查询最近5个交易日的复盘数据

### 4. 结果格式化

脚本返回的 JSON 结果需按以下规则展示给用户：

1. **数据来源标注**：所有结果顶部必须标注：`数据来源：华泰证券研究所`
2. **复盘日期**：必须注明复盘日期
3. **数据完整性**：复盘正文必须完整展示，不得截断
4. **编码**：确保 UTF-8 编码，中文不乱码
5. **风险揭示**：脚本输出末尾固定包含风险揭示，必须完整原样返回，不允许总结或篡改。内容如下：
   > 风险揭示：以上内容由 AI 基于华泰证券研究报告及其附属成果总结生成，可能省略完整报告中的重要信息、市场风险、业绩不及预期等风险，请务必阅读完整原始报告。有关数据、观点、评级等具有时效性，请自行核实最新信息。以上内容仅供研究学习使用，不构成任何投资建议。请勿二次传播并自行承担投资风险。

## 回复格式示例

### 单个交易日复盘示例

```
数据来源：华泰证券研究所

复盘日期：2026-05-18

标题：☕华泰策略 | 每日复盘0518

【市场复盘】
（完整复盘正文内容，不截断）
```

### 多个交易日复盘示例

```
数据来源：华泰证券研究所

=== 第1条 ===
复盘日期：2026-05-18

标题：☕华泰策略 | 每日复盘0518

【市场复盘】
（完整复盘正文内容，不截断）

=== 第2条 ===
复盘日期：2026-05-17
...
```

## 注意事项

- 接口返回的 `review` 字段包含完整的复盘正文，必须完整输出，不得截断
- 所有请求和响应均使用 UTF-8 编码
- 请求头 `Authorization` 的值从环境变量 `HTSC_APP_KEY` 获取
